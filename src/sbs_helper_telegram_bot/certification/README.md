# Модуль аттестации сотрудников

Модуль тестирования знаний сотрудников с рейтингами, историей попыток и режимом обучения.

## Описание

Модуль позволяет проводить тестирование (аттестацию) сотрудников по различным категориям знаний. Поддерживает:

- Тестирование с выбором из 4 вариантов ответа (A/B/C/D)
- Справедливое распределение сложности в тесте (целевой баланс 33%/33%/33% с мягким fallback)
- Ограничение времени на тест
- Категоризацию вопросов с уровнями сложности
- Режим обучения без влияния на рейтинг
- Выбор сложности перед началом обучения
- Ежемесячный рейтинг сотрудников
- Историю всех попыток
- Контроль актуальности вопросов (дата релевантности)
- Настраиваемое отображение правильных ответов
- Анонимизацию имён в рейтинге

## Возможности

### Для пользователей

- **Прохождение тестов** — выборка вопросов с целевым балансом сложности 33%/33%/33%
- **Режим обучения** — тренировка без рейтинга, с показом правильного ответа и комментария
- **Перемешивание вариантов** — порядок ответов A/B/C/D перемешивается каждый раз
- **Выбор категории** — тестирование по конкретной категории или всем сразу
- **Уровни сложности** — 🟢 Лёгкий, 🟡 Средний, 🔴 Сложный
- **Выбор сложности в обучении** — перед началом обучения
- **Единый сертификационный ранг** — ранг формируется по аттестации на основе числа пройденных тестов и освоенных категорий
- **Прогресс в главном меню** — в главном меню показывается прогресс к максимуму очков и визуальный прогресс-бар
- **Срок действия категории** — результат по каждой категории действует 30 дней, затем требует переподтверждения
- **Рейтинг** — личный результат за текущий месяц
- **История** — все предыдущие результаты с деталями
- **ТОП месяца** — лучшие результаты среди всех пользователей (с фильтрацией по категориям)

### Для администраторов

- **Управление категориями** — создание, редактирование, удаление
- **Управление вопросами** — полный CRUD, поиск, просмотр без категории
- **Настройки теста** — количество вопросов, время, проходной балл
- **Отображение ответов** — показ правильных ответов после ответа
- **Анонимизация рейтинга** — скрытие имён в ТОП месяца
- **Контроль актуальности** — просмотр и обновление устаревших вопросов
- **Статистика** — общая статистика тестирования

| questions_count | Количество вопросов в тесте | 20 |

## Расчет рейтинга

1. **Ежемесячный рейтинг** — учитывается лучший результат за месяц
2. **Сортировка** — по проценту правильных (DESC), затем по числу тестов (DESC)
3. **ТОП-10** — отображаются 10 лучших результатов
4. **Анонимизация** — при включении имена скрываются (И... И...)

## Расчет единого ранга аттестации

Единый ранг пользователя строится по результатам тестов по категориям за окно валидности:

- `passed_tests_count` — число успешно пройденных тестов
- `passed_categories_count` — число категорий, в которых есть успешный тест не старше 30 дней
- `certification_points` — сумма лучших процентов по категориям за последние 30 дней
	\(например, лучший результат 100% в категории = 100 очков\)

Порог каждого ранга рассчитывается динамически от максимума достижимых очков
по процентной шкале \(а не фиксированными числами\):

- Новичок — 0%
- Практик — 16%
- Специалист — 36%
- Эксперт — 64%
- Мастер аттестации — 90%
- Абсолют — 100% \(ultimate\)

Максимум достижимых очков рассчитывается по текущему числу активных категорий:

- `max_points = active_categories * 100`.

Полная шкала рангов с порогами баллов показывается пользователю на экране «📊 Мой рейтинг».

Если есть истекшие результаты по категориям, бот явно предупреждает пользователя,
что аттестационный ранг может снизиться.
