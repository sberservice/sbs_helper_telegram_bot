# Модуль обратной связи

Модуль для отправки отзывов и получения ответов от команды поддержки.

## Описание

Модуль позволяет пользователям отправлять категоризированные обращения (ошибки, предложения, вопросы и др.), а администраторам — анонимно отвечать на них. Поддерживает:

- Категоризацию обращений (🐛 Ошибка, 💡 Предложение, ❓ Вопрос, 📝 Другое)
- Отслеживание статусов (новое → в работе → решено / закрыто)
- Анонимные ответы — личность администратора никогда не раскрывается пользователю
- Ограничение частоты — не чаще 1 обращения в час
- Защиту от спама — блокировка сообщений, содержащих ссылки
- Пагинацию списков обращений

## Возможности

### Для пользователей

- **📝 Отправить отзыв** — выбор категории, ввод текста, подтверждение
- **📋 Мои обращения** — список отправленных обращений с пагинацией
- **Просмотр деталей** — статус обращения и ответы от поддержки
- **Защита** — блокировка ссылок (http/https, www, t.me, домены)

### Для администраторов

- **📥 Новые обращения** — список необработанных обращений
- **📊 Все обращения** — полный список с фильтрацией по статусу
- **📂 По категориям** — фильтрация обращений по категориям
- **Ответ** — анонимный ответ пользователю (admin_id не раскрывается)
- **Смена статуса** — изменение статуса обращения (🆕 → ⏳ → ✅ → 🔒)

## Установка

```bash
mysql -u root -p sprint_db < sql/feedback_setup.sql
```

## База данных

### Таблицы

| Таблица | Описание |
|---------|----------|
| `feedback_categories` | Категории обращений |
| `feedback_entries` | Обращения пользователей |
| `feedback_responses` | Ответы администраторов (admin_id — только для внутреннего использования) |

### Начальные категории

| Эмодзи | Категория | Описание |
|--------|-----------|----------|
| 🐛 | Ошибка | Сообщение об ошибке или проблеме |
| 💡 | Предложение | Предложение по улучшению |
| ❓ | Вопрос | Вопрос по работе бота |
| 📝 | Другое | Другой тип обращения |

### Статусы обращений

| Статус | Отображение | Описание |
|--------|-------------|----------|
| `new` | 🆕 Новое | Только что отправлено |
| `in_progress` | ⏳ В работе | Администратор рассматривает |
| `resolved` | ✅ Решено | Проблема решена |
| `closed` | 🔒 Закрыто | Обращение закрыто |

## Меню пользователя

| Кнопка | Действие |
|--------|----------|
| 📝 Отправить отзыв | Создание нового обращения |
| 📋 Мои обращения | Список отправленных обращений |

## Меню администратора

| Кнопка | Действие |
|--------|----------|
| 📥 Новые обращения | Список необработанных обращений |
| 📊 Все обращения | Полный список с фильтрацией |
| 📂 По категориям | Фильтрация по категориям |

## Конвейер обращения

```
Пользователь                           Администратор
    │                                       │
    ├─ Выбор категории                      │
    ├─ Ввод текста                          │
    ├─ Подтверждение ──────────────────────▶ 📥 Новое обращение
    │                                       │
    │                                       ├─ Просмотр
    │                                       ├─ Смена статуса → ⏳ В работе
    │  ◀─────────────────── Анонимный ответ ┤
    │                                       ├─ Смена статуса → ✅ Решено
    │                                       │
    ├─ 📋 Мои обращения                     │
    └─ Просмотр ответа                      │
```

## Ограничения

| Параметр | Значение |
|----------|----------|
| Частота обращений | Не чаще 1 раза в час (3600 сек) |
| Ссылки | Блокируются (http, https, www, t.me, домены) |
| Элементов на странице | 5 |

## Защита от ссылок

Модуль блокирует сообщения, содержащие:
- `http://` или `https://` URL
- Ссылки вида `www.`
- Telegram-ссылки `t.me/`
- Паттерны доменов `example.com`
