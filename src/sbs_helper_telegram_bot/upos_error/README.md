# Модуль UPOS ошибок

Модуль для поиска кодов ошибок системы UPOS с описаниями и рекомендациями по устранению.

## Описание

Модуль предоставляет:

- **Быстрый поиск** кодов ошибок UPOS
- **Подробные описания** и рекомендации по устранению
- **Категоризацию** ошибок
- **Статистику запросов** для анализа проблем
- **Отслеживание** неизвестных кодов
- **Административную панель** для управления базой знаний

## Возможности

### Для пользователей

- 🔍 **Поиск кодов** — введите код и получите описание с рекомендациями
- 📊 **Популярные ошибки** — топ-10 запрашиваемых кодов
- 📁 **Категории** — ошибки организованы по типам
- 📅 **Актуальность** — дата последнего обновления рекомендаций

### Для администраторов

- ➕ Добавление новых кодов ошибок
- ✏️ Редактирование описаний и рекомендаций
- 📥 Импорт из CSV — массовая загрузка кодов
- 📁 Управление категориями
- ❓ Обработка неизвестных кодов
- 📈 Статистика использования
- 🚫 Деактивация/активация кодов
- 🗑️ Удаление ошибок и категорий

## Использование

### Поиск ошибки

1. В меню бота нажмите **🔢 UPOS Ошибки**
2. Введите код ошибки (например, `101` или `E-001`)
3. Получите описание и рекомендации
4. Сразу введите следующий код, если нужно

Примечание: во время ввода кода можно нажать любую кнопку меню или /cancel, чтобы выйти из режима поиска.

Примечание для AI-ввода: если запрос попал в intent `upos_error_lookup`, но код не найден в базе UPOS, бот уведомит об этом и автоматически попробует найти связанную информацию через базу знаний \(RAG\) по исходному тексту запроса.

### Популярные ошибки

Нажмите **📊 Популярные ошибки** для просмотра топ-10 запрашиваемых кодов.

### Импорт из CSV

**Обязательные столбцы:**

| Столбец | Альтернативные названия | Описание |
|---------|-------------------------|----------|
| error_code | код, код_ошибки, code | Код ошибки |
| description | описание, desc | Описание |
| suggested_actions | рекомендации, actions, действия | Рекомендации |

**Опциональные столбцы:**

| Столбец | Описание |
|---------|----------|
| category | Категория (создаётся автоматически) |

**Пример CSV:**

```csv
error_code,description,suggested_actions,category
101,Нет бумаги в принтере,"1. Откройте крышку
2. Установите рулон
3. Закройте крышку",Принтер
102,Замятие бумаги,Извлечь бумагу и перезапустить,Принтер
201,Нет связи с сервером,Проверить интернет,Сеть
```

**Параметры:**
- Кодировка: UTF-8 или Windows-1251
- Разделитель: запятая (,) или точка с запятой (;)
- Максимальный размер: 5 МБ

## FAQ

### Как массово добавить коды?

Используйте импорт CSV или SQL:

```sql
INSERT INTO upos_error_codes (error_code, category_id, description, suggested_actions, created_timestamp) VALUES
('201', 2, 'Нет связи с сервером', 'Проверьте интернет', UNIX_TIMESTAMP());
```

### Что происходит при удалении категории?

Коды ошибок **НЕ удаляются** — их `category_id` становится `NULL`.

### Можно ли деактивировать код вместо удаления?

Да. Код останется в базе, но пользователи его не увидят.
