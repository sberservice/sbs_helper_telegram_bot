# Модуль КТР (Коэффициент Трудозатрат)

Модуль для поиска кодов КТР и получения информации о нормативном времени выполнения работ.

## Описание

КТР — это система кодификации работ с указанием нормативного времени на выполнение. Каждый код содержит:

- **Код** — уникальный идентификатор (например, POS2421)
- **Описание** — название/описание работы
- **Минуты** — нормативное время выполнения
- **Дата обновления** — дата последнего изменения (опционально)
- **Категория** — группировка по типу работ (опционально)

## Возможности

### Для пользователей
- 🔍 Поиск кода КТР по идентификатору
- 📊 Просмотр популярных кодов (топ-10)
- 🎖️ Просмотр достижений за использование модуля (интеграция с геймификацией)
- 📅 Просмотр даты последнего обновления общего файла КТР при входе в модуль
- 📬 Подсказка по отправке более свежего файла КТР через модуль обратной связи

### Для администраторов
- 📋 Просмотр списка всех кодов (с пагинацией)
- 🔍 Поиск кодов
- ➕ Добавление новых кодов
- 📝 Редактирование описания и трудозатрат
- 🗑️ Удаление кодов
- 📁 Управление категориями (создание, редактирование, удаление)
- ❓ Просмотр неизвестных кодов (запрошенных, но отсутствующих в базе)
- 📈 Просмотр статистики использования
- 📥 Импорт кодов из CSV файла
- 📅 Установка/очистка даты последнего обновления общего файла КТР

## Импорт из CSV

### Формат файла

| Столбец | Альтернативные названия | Описание | Обязательный |
|---------|------------------------|----------|--------------|
| code | код, ktr_code, код_ктр | Код КТР | Да |
| description | описание, desc, название | Описание работы | Да |
| minutes | минуты, время, time, мин | Трудозатраты в минутах | Да |
| category | категория, cat | Название категории | Нет |
| date_updated | дата_обновления, дата, date | Дата обновления (дд.мм.гггг) | Нет |

### Пример CSV

```csv
code,description,minutes,category,date_updated
POS2421,Установка POS-терминала,90,POS-терминалы,15.01.2026
POS2422,Замена POS-терминала,60,POS-терминалы,15.01.2026
KASS001,Установка кассы,120,Кассы,
NET001,Настройка роутера,30,Сетевое оборудование,
```

### Рекомендации

- **Кодировка:** UTF-8 или Windows-1251
- **Разделитель:** запятая (,) или точка с запятой (;)
- **Максимальный размер:** 5 МБ

## База данных

### Таблицы

| Таблица | Описание |
|---------|----------|
| `ktr_categories` | Категории кодов КТР |
| `ktr_codes` | Коды КТР с описанием и трудозатратами |
| `ktr_unknown_codes` | Неизвестные коды |
| `ktr_request_log` | Лог запросов для статистики |

### Структура таблицы ktr_codes

| Поле | Тип | Описание |
|------|-----|----------|
| id | bigint | Первичный ключ |
| code | varchar(50) | Уникальный код КТР |
| category_id | bigint | Ссылка на категорию (NULL если не задана) |
| description | text | Описание работы |
| minutes | int | Трудозатраты в минутах |
| date_updated | varchar(10) | Дата обновления минут (дд.мм.гггг) |
| active | tinyint | Статус активности (1/0) |
| created_timestamp | bigint | Unix timestamp создания |
| updated_timestamp | bigint | Unix timestamp обновления |

## Меню пользователя

| Кнопка | Действие |
|--------|----------|
| 📊 Популярные коды | Топ-10 запрашиваемых кодов |
| 🎖️ Достижения | Достижения за использование модуля |

Примечание: во время ввода кода можно нажать любую кнопку меню или /cancel, чтобы выйти из режима поиска.

При входе в модуль пользователю показывается дата последнего обновления общего файла КТР и короткая инструкция: при наличии более свежего файла нужно отправить его через модуль «📬 Обратная связь» с контактами для связи.
